Audio and Mixing

For every project set these specifics

    • Options 🢒 Preferences...
    • tab Preferences
      • Prompt to save on new project → 🗹

    • File 🢒 Project settings
      • tab Project Settings
        • Project sample rate → 48000
      • tab Media
        • On import of media to project → Copy media to project path
        • WAV bit depth → 24 bit PCM
        • Large files → WAV
      • tab Video
        • Frame → 25
      • Save as default project settings

  • When saving the project for the first time, check the following:
    • Create subdirectory for project → 🗹
    • Copy all media into project directory, using: → 🗹

Additional notes

  • The preferred buffer is set to 512 samples (as seen in class, or 128 samples as seen in the pdf prof Martini shared) to achieve lower latency. More samples may be needed for >10 tracks
  • The ruler's unit can be changed by clicking with the right mouse button on the Arrangement Window
  • Every track has basic controls, such as volume, fase, mute, solo, ...
  • Master controls and the track controls are linked
  • The mixer can be docked and undocked through the rmb contextual menu
  • Windows doesn't allow to drag and drop files to the Arrangement Window from a preview of a zip folders content

Definitions and names

  • Item → audio/midi/video track
  • MIDI → Musical Instrument Digital Interface
  • Arrangement Window → the window where the audio tracks are visually displayed

Shortcuts

ctrl + M → Show/hide mixer
W → Place the head/cursor to the start of the timeline
End → Place the head/cursor to the end of the timeline
alt + Enter → Open Project Settings
ctrl + P → Open Preferences
ctrl + alt + T → Show/hide Transport (Play/Record/Stop...)
